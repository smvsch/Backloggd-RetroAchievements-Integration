# Backloggd-RetroAchievements-Integration
Integrates RetroAchievements stats directly into Backloggd profiles and game pages.
## This is a work-in-progress and was mainly the result of a spontaneous hyperfixation
### Features:

- Automatically fetches and displays your achievement progress for the game you are viewing.

- Allows you to link your Backloggd account to your RetroAchievements profile, enabling friends to see your RA stats (Score, Global Rank, and Trophy Showcase) directly on your Backloggd profile.

- Uses a dedicated backend proxy to handle database communications, ensuring no secret keys are exposed in the client-side code and your data remains private.

###How to Use:

- Navigate to any game page on Backloggd.

- Enter your RetroAchievements Username and Web API Key when prompted.
